# 数据抓取并写入Excel
  ##  流程逻辑
  - 自动打开教学环境网站 批量抓取表格数据 写入Excel  
    ![alt text](<assets/Data scraping/image-2.png>)

  ## 操作步骤
  1. 打开网页：打开教学网站网页
    ![alt text](<assets/Data scraping/image-3.png>)
  2. 点击数据抓取按钮
    ![alt text](<assets/Data scraping/image-4.png>)
  3. 点击下一步
    ![alt text](<assets/Data scraping/image-5.png>)
  4. 按ctrl+左键 捕捉一个单元格元素
    ![alt text](<assets/Data scraping/image-6.png>)
  5. 抓取元素后 可预览抓取内容 以及自定义抓取页数 如需要拾取多页点击是 不需要点击否即可
    ![alt text](<assets/Data scraping/image-7.png>)
  6. 点击是确定抓取多页后 需要再捕捉 下一页 按钮 完成抓取
    ![alt text](<assets/Data scraping/image-8.png>)
  7. 新建或打开Excel：打开要写入的Excel文件
    ![alt text](<assets/Data scraping/image-9.png>)
  8. 写入数据至Excel工作表：将抓取到的数据写入Excel
    ![alt text](<assets/Data scraping/image-10.png>)
