# 准备操作
   - 下载示例应用脚本 以及测试票据图片 并导入脚本  
    1. [点击下载示例脚本](https://download.krpalite.com:56780/Example/Poc基础操作.zip)  
    2. 解压下载的压缩包  
    3. 在lite首页 点击导入按钮  
      ![alt text](<assets/Get ready/image-3.png>)
    4. 选择解压出来的脚本文件 POC操作示例.krpa  
      ![alt text](<assets/Get ready/image-4.png>)
    5. 运行 打开网页 流程 进入lite官网  
      ![alt text](<assets/Get ready/image-2.png>)
    6. 运行 进入教学环境 流程 打开教学环境网站
      ![alt text](<assets/Get ready/image-1.png>)
  ##  注意点
  - excel读写流程需要修改‘打开或新建Excel’组件中的excel路径
    ![alt text](<assets/Get ready/image-9.png>)
    ![alt text](<assets/Get ready/image-6.png>)

  - 票据识别流程需要修改‘票据识别’组件中的票据图片路径
