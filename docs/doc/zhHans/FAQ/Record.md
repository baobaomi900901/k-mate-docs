# 屏幕录制操作

## 准备操作

1. [点击下载屏幕录制插件](https://download.krpalite.com:56780/TaskRecScreen.zip)

2. 把下载的压缩包在lite同目录下解压
   
  ![alt text](assets/Record/image-1.png)

  ![alt text](assets/Record/image-2.png)

3. 开启屏幕录制功能
   
  ![alt text](assets/Record/image.png)

4. 定时任务执行时可自动录制
   
  ps：<span style="color:red;">流程执行时间<=10秒 无法录制</span>
