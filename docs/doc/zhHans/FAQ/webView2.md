## 原因

用户的系统缺少 Microsoft WebView2 运行环境. 这在一些企业定制的 windows 系统中是非常常见的现象.

:::tip WebView2 是什么?

WebView2 是微软推出的一个控件，允许开发者在 Windows 应用程序中嵌入基于 Chromium 的浏览器引擎，用于渲染网页内容。它是 Internet Explorer 的 WebView 控件的继任者，基于更现代的 Edge 浏览器技术。

:::
