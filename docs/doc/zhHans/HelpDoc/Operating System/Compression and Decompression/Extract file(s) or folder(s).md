# 解压文件(夹)
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
将压缩包中的内容进行解压
:::

![解压文件(夹)](../../../assets/解压文件(夹)_command.png)

## 配置项说明

### 指令输入

- **压缩文件路径**`string`: 
  - 输入或选择要解压的文件路径

- **密码**`string`: 
  - 输入密码，若无密码则留空

- **解压文件夹路径**`string`: 
  - 请输入解压输出文件夹路径

- **创建同名文件夹**`string`: 
  - 解压至与压缩文件同名的文件夹中

- **操作范围**`Integer`: 
  - 选择覆盖会覆盖现有同名文件而不询问；选择跳过会跳过提取同名的文件；选择保存时如果文件同名，自动重命名解压出的文件

- **执行前的延迟(毫秒)**`string`: 
  - 指令执行前的等待时间

### 指令输出

- **保存文件夹路径至**`string`: 
  - 指定一个变量名称，该变量用于保存文件夹路径
