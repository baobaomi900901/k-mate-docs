# 删除重复行
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
删除数据表格对象的指定列或整个表的重复行数据
:::

![alt text](<assets/Delete Duplicate Rows/image.png>)

## 配置项说明

### 指令输入

- **数据表格对象**`TDataTable`: 
  - 可以使用“创建数据表格”组件返回的变量

- **删除方式**`Integer`: 
  - 选择删除方式

- **指定列名**`TList<string>`: 
  - 所选列数据都一致时则删除。单列输入["列名1"]，多列输入["列名1","列名2","列名3"]

- **删除模式**`Integer`: 
  - 检测到重复行时，可选择保留首行、保留尾行或全部删除

- **在数据表中预览**`Boolean`: 
  - 在数据表中预览

- **打印组件的输出变量值**`Boolean`: 
  - 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中

### 指令输出

- **删除后数据**`Integer`: 
  - 用于存储删除重复后的数据表格
