# 按列转换数据类型
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
将数据表格对象的指定列或整个表转换为字符串(str)、整数(int)、浮点数(float)
:::

![alt text](<assets/Convert data types by column/image.png>)

## 配置项说明

### 指令输入

- **数据表格对象**`TDataTable`: 
  - 可以使用“创建数据表格”组件返回的变量

- **指定列名**`TList<string>`: 
  - 输入需要转换数据类型的列，单列输入["列名1"]，多列输入["列名1","列名2","列名3"]

- **转换为**`Integer`: 
  - 输入需要转换为何种数据类型

- **在数据表中预览**`Boolean`: 
  - 在数据表中预览

- **打印组件的输出变量值**`Boolean`: 
  - 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中

- **忽略首行**`Boolean`: 
  - 勾选后，将忽略数据表格的首行数据进行转换

### 指令输出

- **转换后结果**`string`: 
  - 转换后结果
