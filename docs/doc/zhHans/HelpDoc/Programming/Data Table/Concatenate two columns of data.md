# 拼接两列数据
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
将两列数据按指定的连接符拼接成新的列
:::

![alt text](<assets/Concatenate two columns of data/image.png>)

## 配置项说明

### 指令输入

- **数据表格对象**`TDataTable`: 
  - 可以使用“创建数据表格”组件返回的变量sdf。

- **待拼接列名1**`string`: 
  - A或1代表第一列, 支持列头名，支持负数，-n表示倒数第n列

- **待拼接列名2**`string`: 
  - A或1代表第一列, 支持列头名，支持负数，-n表示倒数第n列

- **连接符**`string`: 
  - 输入两列数据拼接的连接符。为空时将直接拼接。

- **在数据表中预览**`Boolean`: 
  - 在数据表中预览

- **打印组件的输出变量值**`Boolean`: 
  - 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中

- **忽略首行**`Boolean`: 
  - 勾选后，将忽略数据表格的首行数据进行转换

- **新增列列名**`string`: 
  - 指定列名或新增，两列拼接之后显示的列名，为空时表示列名也参与拼接。
  
### 指令输出

- **拼接后数据**`Integer`: 
  - 将两列数据按指定的连接符拼接成新的列。
