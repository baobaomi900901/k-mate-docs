# 读取数据表格内容
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
读取数据表格中的部分数据
:::

![alt text](<assets/Read the data table content/image.png>)

## 配置项说明

### 指令输入

  - **数据表格对象**`TDataTable`: 可使用“创建数据表格”组件返回的变量。

  - **读取方式**`Integer`: 选择按指定的行名、列名读取数据，或按指定的行号、列号读取数据

  - **行号**`string`: 输入一个要查单元格的行号

  - **列名**`string`: 输入一个要查单元格的列名

  - **指定行号**`string`: 输入一个要查数据的行号

  - **指定列名**`string`: 输入一个要查单元格的列名

  - **选择模式**`Integer`: 普通模式：需要输入起始/结束，行号/列名  表达式模式：可输入范围表达式 例如1:3

  - **起始行号**`Integer`: 输入一个要查范围的起始行号

  - **结束行号**`Integer`: 输入一个要查范围的结束行号

  - **起始列名**`string`: 输入一个要查范围的起始列名

  - **结束列名**`string`: 输入一个要查范围的结束列名

  - **按读取区域输出**`string`: 读取区域为单元格时输出为字符串，为行或列时输出为列表，为区域时输出为数据表格

  - **在数据表格中浏览**`string`: 勾选后可在数据表格查看表格数据

  - **打印组件的输出变量值**`TDataTable`: 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中


### 指令输出

  - **读取结果保存到**`TDataTable`: 指定一个变量，用于保存读取结果
