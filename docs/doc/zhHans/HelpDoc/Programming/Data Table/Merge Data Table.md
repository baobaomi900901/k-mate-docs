# 合并数据表
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
将指定数据表与当前数据表合并，指定是否保留更改以及如何处理当前数据表中缺少的架构。
:::

![alt text](<assets/Merge Data Table/image.png>)

## 配置项说明

### 指令输入

- **目标**`string`: 
  - 要将源数据表合并到的数据表对象。

- **来源**`string`: 
  - 要添加到目标数据表的数据表对象。

- **缺少架构操作**`Integer`: 
  - 缺少架构操作

- **在数据表中预览**`Boolean`: 
  - 在数据表中预览

- **打印组件的输出变量值**`Boolean`: 
  - 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中

### 指令输出

- **输出数据表**`Integer`: 
  - 生成的已合并的数据表变量。
