# 写入内容至数据表格
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
在数据表格写入内容，支持追加、插入或覆盖
:::



## 配置项说明

### 指令输入

- **数据表格对象**`TDataTable`: 
  - 可以使用“创建数据表格”组件返回的变量。

- **写入范围**`Integer`: 
  - 将数据写入到单元格、区域、一行或一列

- **行号**`Integer`: 
  - 行号默认为数据表格每一行的首位

- **列名**`string`: 
  - A或1代表第一列, 支持列头名，支持负数，-n表示倒数第n列

- **写入方式**`Integer`: 
  - 提示

- **起始列名**`string`: 
  - A或1代表第一列, 支持列头名，支持负数，-n表示倒数第n列

- **起始行号**`Integer`: 
  - 行号默认为数据表格每一行的首位

- **写入内容**`string`: 
  - 写入内容

- **在数据表中预览**`Boolean`: 
  - 在数据表中预览

- **打印组件的输出变量值**`Boolean`: 
  - 勾选后，将组件运行产生的变量数据或变量值输出，并打印到控制台输出日志中

- **是否清空原表列头**`Boolean`: 
  - 勾选后，将清空原表列头

### 指令输出

- **写入后结果**`string`: 
  - 输出写入数据成功后的数据表格。
