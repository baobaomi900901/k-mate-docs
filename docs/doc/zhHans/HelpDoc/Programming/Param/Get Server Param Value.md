# 获取全局参数

## 功能说明

:::tip 功能描述
获取保存在服务器中的全局参数值
:::

![alt text](<assets/Get Server Param Value/image.png>)

## 配置项说明

### 常规

**指令输入**

- **参数名称**`string`: 
  
  - 需要获取的全局参数名称

**指令输出**

- **将全局参数保存到**`string`:  
  
  - 指定一个用来保存全局参数值的变量


## 使用示例

使用[设置全局参数]函数 来创建一个全局参数 完成后可用[获取全局参数]函数 获取到此参数
![alt text](<assets/Get Server Param Value/image-1.png>)
