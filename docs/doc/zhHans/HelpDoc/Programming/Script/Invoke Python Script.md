# 调用 Python

- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
执行指定的 Python 脚本，可选择通过文件路径或直接输入脚本内容的方式调用，并可向其传递一个输入参数列表。
:::

## 配置项说明

### 常规

**指令输入**

- **参数** `string`：传递给 Python 脚本的参数列表。此项必须为集合的形式，如 `[xxx, 'xxx']`，列表内的参数须为 `string` 类型。若无需传递参数，则无需填写。

- **传入方式** `Enum`：选择调用 Python 脚本的方式，支持以下两种模式：
  - **文件**：通过指定脚本文件路径进行调用。
    - **文件路径** `string`：要调用的 Python 脚本文件的完整路径。

  - **脚本**：通过直接输入脚本内容进行调用。
    - **脚本** `string`：要调用的 Python 脚本内容。
    - **输入编码格式** `Enum`：脚本内容的编码格式。可选值：`Default` / `Ansi` / `UTF8` / `UTF16` / `UTF32`。

- **执行的类名** `string`：要调用的 Python 脚本中的类名。

- **执行的方法名** `string`：要调用的 Python 脚本中的方法名。

- **输出编码格式** `Enum`：脚本标准输出的编码格式。可选值：`Default` / `Ansi` / `UTF8` / `UTF16` / `UTF32`。

- **错误编码格式** `Enum`：脚本错误输出的编码格式。可选值：`Default` / `Ansi` / `UTF8` / `UTF16` / `UTF32`。

**指令输出**

- **结果** `string`：Python 脚本的执行结果（对应变量 `_Result`）。

- **错误信息** `string`：Python 脚本执行过程中产生的错误信息（对应变量 `ErrorInfo`）。

### 错误处理

- **打印错误日志** `Boolean`：当指令运行出错时，打印错误日志到【日志】面板。默认勾选。

- **处理方式** `Integer`：
  - **终止流程**：指令运行出错时，终止流程。

  - **忽略异常并继续执行**：指令运行出错时，忽略异常，继续执行流程。

  - **重试此指令**：指令运行出错时，重试运行指定次数指令，每次重试间隔指定时长。
