# 消息通知
- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
用于打开或关闭消息提示框
:::

![消息通知](../../../assets/消息通知_command.png)

## 配置项说明

### 指令输入

- **操作**`Integer`: 
  - 可选择打开或者关闭消息通知框

- **消息类型**`Integer`: 
  - 提示框展示的消息类型，可选择：信息、警告、错误

- **消息位置**`Integer`: 
  - 提示框展示时相对屏幕的位置，可选择：顶部、底部、右下角

- **消息内容**`string`: 
  - 提示框要展示给用户的内容

- **关闭的超时时间(毫秒)**`string`: 
  - 通知信息展示的时长，默认为3000毫秒

### 指令输出

  - 当前组件不包含任何输出项
