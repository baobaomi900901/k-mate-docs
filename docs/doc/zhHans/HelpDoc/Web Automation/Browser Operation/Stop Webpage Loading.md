# 停止网页加载

## 功能说明

:::tip 功能描述
当网页未加载完毕时，则强制停止网页加载
:::

![停止网页加载](../../../assets/停止网页加载_command.png)

## 指令输入

- **网页对象**`TBrowser`: 
    - 输入一个获取到的或通过'打开网页'函数创建的网页对象

### 高级

- **执行前的延迟(毫秒)**`Integer`: 
    - 指令执行前的等待时间（毫秒）

## 使用示例

![停止网页加载](../../../assets/停止网页加载_demo.png)

**流程逻辑描述：** 使用【打开网页】指令打开指定网页 --> 使用【停止网页加载】指令强制停止网页加载
