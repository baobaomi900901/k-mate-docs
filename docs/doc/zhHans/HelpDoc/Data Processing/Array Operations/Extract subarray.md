# 截取子数组
- 适用系统: windows

## 功能说明

:::tip 功能描述
从一个数组中截取一部分内容输出一个子数组
:::

![alt text](<assets/Merge arrays copy/image-1.png>)
## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量
  
  **开始截取位置**`Integer`: 
  - 需要截取的起始索引
  
  **结束截取的位置**`Integer`: 
  - 需要截取的结束索引
  
## 指令输出

  **保存结果至**`Array`: 
  - 返回一个截取后的数组

## 使用示例

![alt text](<assets/Merge arrays copy/image-2.png>)
  - 使用【设置变量】定义数组变量 --> 使用【截取子数组】截取下来一段数组 --> 打印输出截取后的数组
