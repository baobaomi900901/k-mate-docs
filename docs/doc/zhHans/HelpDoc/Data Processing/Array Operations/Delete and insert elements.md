# 删除并插入元素
- 适用系统: windows

## 功能说明

:::tip 功能描述
可以对数组进行删除，替换，插入操作，删除数量为0则可仅插入，插入数组为空
则可仅删除
:::

![alt text](<assets/Extract subarray copy/image-2.png>)

## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量
  
  **开始位置**`Integer`: 
  - 从什么索引位置开始删除和插入操作
  
  **删除数量**`Integer`: 
  - 需要删除元素的数量
  
  **插入数组**`Array`: 
  - 输入获选择一个需要插入的数组
  
## 指令输出

  **保存结果至**`Array`: 
  - 返回一个更改后的数组
