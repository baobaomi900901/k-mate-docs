# 数组转字符串
- 适用系统: windows

## 功能说明

:::tip 功能描述
将数组转为字符串 并通过设定的分隔符进行分割
:::

![alt text](<assets/Array to String/image-1.png>)

## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量

  **分隔符**`String`: 
  - 自定义分割符 为空时 默认为逗号(,)


## 指令输出

  **转换结果**`string`: 
  - 数组转为字符串后的结果

## 使用示例

  ![alt text](<assets/Array to String/image.png>)

  - 使用【设置变量】定义一个数组变量 --> 使用【数组转字符串】将数组转为字符串 --> 打印输出字符串
