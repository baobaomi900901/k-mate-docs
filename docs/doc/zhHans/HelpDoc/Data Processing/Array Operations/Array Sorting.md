# 数组排序
- 适用系统: windows

## 功能说明

:::tip 功能描述
将数组通过降序/升序重新排列 中文按拼音首字母 英文按首字母 数字按大小
:::

![alt text](<assets/Array Sorting/image.png>)

## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量

  **排序方式**`Integer`:
  - 选择一个排序方式 升序/降序

## 指令输出

  **保存结果至**`Array`: 
  - 指定一个保存排序后的数组
