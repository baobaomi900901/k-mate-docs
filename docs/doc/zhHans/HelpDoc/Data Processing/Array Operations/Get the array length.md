# 获取数组长度
- 适用系统: windows

## 功能说明

:::tip 功能描述
获取数组的长度
:::

![alt text](<assets/Get the array length/image.png>)

## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量

## 指令输出

  **转换结果**`Integer`: 
  - 指定一个保存数组长度的变量

## 使用示例

  ![alt text](<assets/Get the array length/image-1.png>)

  - 使用【设置变量】定义一个数组变量 --> 使用【获取数组长度】来获取数组的长度 --> 打印输出数组长度
