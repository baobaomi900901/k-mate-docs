# 开头/末尾添加元素
- 适用系统: windows

## 功能说明

:::tip 功能描述
在数组开头/末尾添加一个元素
:::

![alt text](<assets/Add elements to the beginning or end/image-1.png>)
## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量

  **元素**`Array`: 
  - 输入一个要添加的元素 字符串需要用''单引号
## 指令输出

  **保存结果至**`Array`: 
  - 指定一个添加元素后的数组

## 使用示例

![alt text](<assets/Add elements to the beginning or end/image-2.png>)

  - 使用【设置变量】定义一个数组变量 --> 使用【开头/末尾添加元素】添加一个元素 --> 打印输出添加后的数组
