# 合并数组
- 适用系统: windows

## 功能说明

:::tip 功能描述
将两个数组合并在一起
:::

![alt text](<assets/Merge arrays/image-1.png>)
## 指令输入

  **待合并数组1**`Array`: 
  - 待合并数组1
  
  **待合并数组2**`Array`: 
  - 待合并数组2
  
## 指令输出

  **保存结果至**`Array`: 
  - 返回一个合并后的数组

## 使用示例

![alt text](<assets/Merge arrays/image.png>)
  - 使用【设置变量】定义两个数组变量 --> 使用【合并数组】将两个数组合并在一起 --> 打印输出合并后的数组
