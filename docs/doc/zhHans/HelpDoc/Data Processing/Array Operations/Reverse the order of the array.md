# 反转数组顺序
- 适用系统: windows

## 功能说明

:::tip 功能描述
将数组元素顺序反转
:::

![alt text](<assets/Reverse the order of the array/image.png>)

## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量

## 指令输出

  **保存结果至**`Array`: 
  - 指定一个保存反转后的数组

## 使用示例

  ![alt text](<assets/Reverse the order of the array/image-1.png>)

  - 使用【设置变量】定义一个数组变量 --> 使用【反转数组顺序】反转数组顺序 --> 打印输出反转后的数组
