# 移除开头/末尾元素
- 适用系统: windows

## 功能说明

:::tip 功能描述
移除并返回开头/末尾的一个元素
:::

![alt text](<assets/Remove the first or last element/image.png>)
## 指令输入

  **数组**`Array`: 
  - 输入或选择一个待转换的数组变量
  - 
## 指令输出

  **保存移除的元素至**`Array`: 
  - 返回被移除的元素
  
  **保存结果至**`Array`: 
  - 指定一个移除元素后的数组
