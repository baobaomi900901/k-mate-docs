# ForEach字典循环

## 功能说明

:::tip 功能描述
该指令用于依次循环字典中的每一对键值对
:::

## 配置项说明

### 常规

**指令输入**

- **字典**`TRPADictionary`: 输入字典变量，可通过函数[新建字典]获取


**指令输出**

- **键名**`string`: 指定一个变量名称，用于保存当前循环的键名

- **键值**`string`: 指定一个变量名称，用于保存当前循环的键值
