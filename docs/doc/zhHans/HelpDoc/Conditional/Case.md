# Case

## 功能说明

:::tip 功能描述
Case多分支选择结构，用于根据一个表达式的不同值执行不同的代码块,可使用整型，字符串进行判断，字符串需要使用单引号括起来进行区分 说明
:::

![alt text](assets/Case/image.png)

## 常规

### 指令输入

**条件**:
  设置一个初始对照值 用于Case Value判断

## 使用示例

![alt text](assets/Case/image-1.png)

**流程逻辑描述：**:
  使用Case函数 设置一个对照值 配合Case Value函数 进行匹配并执行匹配成功的代码块
