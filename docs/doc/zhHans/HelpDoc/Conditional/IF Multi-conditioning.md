# IF 多条件

- 适用系统: windows / 信创

## 功能说明

:::tip 功能描述
作为条件判断的开始标记
:::

![alt text](<assets/IF Multi-conditioning/image-1.png>)

## 指令输入

- **条件关系**:
  - 下面使用的条件将会以此关系进行合并得到最终的结果
- **条件列表**
  - 可添加多个条件
