# 获取当前激活的Excel对象

## 功能说明

:::tip 功能描述
获取当前激活的Excel对象。若有多个Excel文件处于打开状态，则获取最近一次操作过的Excel
:::

![获取当前激活的Excel对象](../../../assets/获取当前激活的Excel对象_command.png)

## 指令输出

- **存储Excel对象到**`TWorkbookApplication`: 
    - 保存获取到的Excel对象，后续流程需要用到该Excel表格时，可以直接调用该变量

## 高级

- **执行前的延迟(毫秒)**`Integer`: 
    - 指令执行前的等待时间
