# 写入单元格

## 功能说明

:::tip 功能描述
将值/公式覆盖写入电子表格中的一个单元格或范围内，更改后会立即保存
:::

![写入单元格](../../../../assets/写入单元格_command.png)

## 指令输入

- **Excel对象**`TWorkbookApplication`: 
  - 输入一个通过函数'打开或新建Excel'/'获取当前激活的Excel对象'存储的Excel对象

- **范围**`string`: 
  - 填写单元格（如`A1`）或范围（如`A1:F4`）

- **工作表名称**`string`: 
  - 工作簿中的工作表名称

- **值**`string`: 
  - 输入值(如`123`)或公式(如`=A1+A2`)
