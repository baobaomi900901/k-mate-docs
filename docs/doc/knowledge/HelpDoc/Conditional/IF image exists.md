# IF 图像存在

## 功能说明

:::tip 功能描述
判断目标图像是否存在
:::

## 配置项说明

### 常规

**指令输入**

- **全部图像存在(不存在)后执行**`Boolean`: 检测所有图像或者检测其中一张图像

- **目标图像**`string`: 可从图像库选择多张图像

- **sAllImages**`string`: 

- **判断条件**`Integer`: 指定查找范围内指定的图像存在或者不存在

- **窗口对象**`TWinObj`: 选择窗口对象

- **搜索范围**`Integer`: 可选择整个屏幕或者指定窗口


**指令输出**

当前指令无输出
