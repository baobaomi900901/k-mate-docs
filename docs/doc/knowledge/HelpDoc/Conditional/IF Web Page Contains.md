# IF 网页包含

## 功能说明

:::tip 功能描述
检查指定的文本或元素是否出现在网页上
:::

## 配置项说明

### 常规

**指令输入**

- **文本内容**`string`: 选择要检查的文本内容

- **操作目标**`TTarget`: 选择要检查的网页元素

- **检测网页是否**`Integer`: 指定你想要检测的内容类型，检查元素或者检查文本

- **网页对象**`TBrowser`: 输入一个获取到的或者通过'打开网页'函数创建的网页对象


**指令输出**

当前指令无输出
